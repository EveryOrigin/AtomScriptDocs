# AtomScript 开发文档

---

### 介绍

欢迎来到AtomScriptDocs,这里提供了关于实际脚本开发的指导教程与建议。

AtomScript基于TabooLib 6.0 开发，使用Jython作为双语言沟通框架。

这意味着脚本可以双向使用Python与Java作为开发语言。

同时因为插件自身框架采用TabooLib，跨版本与平台特性也具备(跨平台未实测)。

本文档基于 Docsify 开发
